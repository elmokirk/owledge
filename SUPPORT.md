# Support

## Where To Start

- Usage and setup questions: open a GitHub issue.
- Documentation gaps: open a GitHub issue with the affected page and expected guidance.
- Security issues: follow [SECURITY.md](SECURITY.md) and do not use public issues for undisclosed findings.

## Good Support Requests

Include:

- your runtime or harness
- your operating system
- the command you ran
- the relevant local path shape
- the validation or gate output that failed

## Out Of Scope

This repository does not provide a hosted service, customer data recovery, or enterprise operations support.
