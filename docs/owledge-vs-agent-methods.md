# Owledge Vs Agent Methods

Owledge is not trying to be every kind of AI coding repo.

## Owledge Vs Superpowers

Superpowers is an execution methodology for coding agents: planning, implementation structure, subagent flow, and review discipline.

Owledge is the persistent Markdown layer around that work:

- durable plans
- evidence paths
- handoffs
- reviews
- session continuity
- existing knowledgebase integration

Short version: Superpowers helps agents execute well. Owledge helps projects remember well.

## Owledge Vs Ponytail

Ponytail is about reducing code and forcing leaner implementation choices.

Owledge is about making project context durable and reviewable:

- what was decided
- why it was decided
- what evidence supported it
- where the next agent should continue

Short version: Ponytail cuts bloat in implementation. Owledge cuts context loss in delivery.

## Why Owledge Exists

If your repo or vault already contains real notes, real plans, real handoffs, and work spread across multiple sessions, you need a durable memory layer more than another chat-only workflow.
