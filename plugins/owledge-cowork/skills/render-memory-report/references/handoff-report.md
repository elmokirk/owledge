# Handoff Report

Use for PM to dev, dev to QA, agent to agent, orchestrator to worker, or client handoff requests.

Sources: `handoffs/`, `sessions/`, `compiled/`, `evidence/`, task cards, QA reports.

Sections: objective, current state, open decisions, blockers, write scope, validation, sources.

Visual pattern: owner cards, checklist bands, risk callouts, and validation command blocks.
