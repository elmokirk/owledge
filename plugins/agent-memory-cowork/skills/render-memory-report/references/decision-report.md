# Decision Report

Use for ADRs, architecture decisions, strategic tradeoffs, implementation choices, or user requests like "show this decision visually".

Sources: `agent-memory/decisions/`, `canonical/`, `evidence/`, `compiled/`.

Sections: decision summary, context, options compared, chosen path, rejected options, risks, evidence, follow-up actions.

Visual pattern: comparison tables, risk badges, status chips, and timeline blocks.
