# PI Proof Kit

PI Proof Kit is an optional add-on that demonstrates a complete persistent
intelligence loop:

```text
observe -> detect recurring-error and parallel -> red-team -> promoted-pattern
and lesson -> measure improvement
```

It is intentionally synthetic. Its purpose is to prove the workflow shape before
teams run PI Agent workflows on real projects.

## Install

```bash
python tools/owledge.py install-addon --project-root . --addon pi-proof-kit
```

## Launch Gate Relevance

The launch-readiness gate checks that this add-on includes at least ten proof
corpus files and a non-empty red-team scorecard with a non-zero score.
