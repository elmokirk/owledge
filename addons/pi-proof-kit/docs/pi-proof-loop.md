# PI Proof Loop

The PI proof corpus closes one learning loop:

1. Observe repeated context-loss and handoff gaps.
2. Detect a recurring-error and a parallel across launch and runtime work.
3. Red-team the proposed interpretation.
4. Promote a reusable pattern and lesson.
5. Measure whether recurrence and launch score improved.

The proof files are synthetic and safe for demos. They should not be promoted
into shared production memory without curator review.
