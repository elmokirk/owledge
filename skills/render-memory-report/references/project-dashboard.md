# Project Dashboard

## Use When

Use for stakeholder summaries, project status reports, executive updates, and non-technical readouts.

## Sources

Prefer:

- `PROJECT_CONTEXT.md`
- `agent-memory/canonical/`
- `agent-memory/compiled/`
- `agent-memory/decisions/`
- `agent-memory/lessons/`

## Sections

| Section | Purpose |
| --- | --- |
| Project status | Current health and focus |
| Goals | Strategic objective |
| Workstreams | Main active areas |
| Decisions | Important choices |
| Risks | Open concerns |
| Next actions | Clear owner-oriented steps |
| Sources | Links and hashes |

## Visual Pattern

Use a concise dashboard layout with cards, tables, and readable status chips.
