# Agent Activity Report

## Use When

Use when the user asks what agents did, how delegation progressed, how many sessions were captured, or how work moved through a multi-agent workflow.

## Sources

Prefer:

- `agent-memory/sessions/`
- `agent-memory/compiled/`
- `agent-memory/evidence/`
- `agent-memory/handoffs/`
- simulation or eval reports when present

## Sections

| Section | Purpose |
| --- | --- |
| Activity summary | Agents, sessions, events, drafts |
| Delegation map | Parent/child or level view |
| Outputs | What was produced |
| Risks | Failures, blockers, missing QA |
| Promotion candidates | Drafts worth curator review |
| Sources | Links and hashes |

## Visual Pattern

Use timelines, swimlanes, agent cards, and throughput metrics. Do not expose raw prompts in shared reports.
