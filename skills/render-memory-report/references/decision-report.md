# Decision Report

## Use When

Use for ADRs, architecture decisions, strategic tradeoffs, implementation choices, or user requests like "show this decision visually".

## Sources

Prefer:

- `agent-memory/decisions/`
- `agent-memory/canonical/`
- `agent-memory/evidence/`
- `agent-memory/compiled/`

## Sections

| Section | Purpose |
| --- | --- |
| Decision summary | One-screen answer |
| Context | Why the decision exists |
| Options compared | Alternatives and tradeoffs |
| Chosen path | What was selected |
| Rejected options | Why they were rejected |
| Risks | Known failure modes |
| Evidence | Source files and hashes |
| Follow-up actions | What agents should do next |

## Visual Pattern

Use comparison tables, risk badges, status chips, and timeline blocks. Keep source links visible.
