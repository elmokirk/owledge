# Proof Add-ons Instead Of Core Expansion

The selected approach keeps Owledge as a memory/proof layer and adds capabilities through optional installable kits.

This directly addresses the adoption concern: projects can keep their existing instructions, hooks, Graphify-style tools, LLM wikis, Obsidian vaults, and agent runtimes.
